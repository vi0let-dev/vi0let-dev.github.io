机器学习-环境配置

本用于搭建从零开始搭建一套完整的机器学习开发环境，涵盖 Python 环境管理、深度学习框架、GPU 支持及常用开发工具配置。

环境概述

机器学习开发通常需要以下组件：

  • Python 运行环境
  • 包管理工具（pip / conda）
  • 深度学习框架（PyTorch）
  • GPU 加速支持（CUDA + cuDNN）
  • 开发工具（VScode/Pycharm）

python运行环境

由于不同的项目的python和cuda环境有所不同，推荐安装Miniconda

下载Miniconda

官方下载地址：https://www.anaconda.com/download

建议选择Miniconda版本(自定义安装，前者预装的第三方库太多)

若官网无法访问或下载太慢，可以选择清华镜像源进行下载

清华镜像源下载地址：https://mirrors.tuna.tsinghua.edu.cn/anaconda/

注意：archive存放的是anaconda版本，miniconda是存放的后者的版本，按需下载即可

安装步骤

Windows：

  1. 双击安装包
  2. 选择 “Add Anaconda to PATH”（建议勾选）
  3. 安装完成

建议软件不要安装在系统盘(占用空间较多)

深度学习框架

从PyTorch 2.7版本开始，通过pip 或 conda安装的官方预编译包已经内置了CUDA运行时和cuDNN 库，因此大多数情况下无需再手动下载和配置cuDNN。

pytorch官网：https://pytorch.org/get-started/locally/

根据python和cuda版本选择合适的pytorch版本，页面如下

可以在上面选择老版本进行指令安装，也可以通过观察新版本的安装指令pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu132最后的cu132是cuda的版本号13.2，你想安装版本为11.8对于的pytorch环境，可以修改为pip3 install torch torchvision --index-url https://download.pytorch.org/whl/cu118

注意右上角的cuda版本是支持的最高版本而不是一定需要安装这个版本，若该支持最高的版本较低请更新驱动。
